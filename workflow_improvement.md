# Monthly Marketing Report Workflow

## Goal

Create a monthly report that explains what happened, why it happened, and what to do next.

## Sections

### 1. Executive summary

- What improved
- What got worse
- Biggest reason
- Main decision needed

### 2. KPI table

| Metric | Previous month | Current month | Change | Comment |
|---|---:|---:|---:|---|
| Spend |  |  |  |  |
| Leads |  |  |  |  |
| CPL |  |  |  |  |
| Qualified leads |  |  |  |  |
| Sales |  |  |  |  |
| Revenue |  |  |  |  |
| ROMI |  |  |  |  |

### 3. Channel analysis

For each channel:

- spend
- leads
- quality
- sales
- problems
- next action

### 4. Funnel analysis

- traffic
- clicks
- leads
- qualified leads
- calls
- sales
- repeat purchases

### 5. CRM quality

Check:

- missed leads
- no source
- no next task
- old deals
- unclear lost reasons

### 6. Recommendations

Rank by:

- impact
- effort
- deadline
- owner

### 7. Next month plan

3-5 actions only.
