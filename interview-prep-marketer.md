# CRM Implementation Prompt

## Goal

Create a clear CRM implementation plan for a small team.

## Prompt

You are a CRM and sales operations consultant.

Build a CRM implementation plan based on the information below.

## Context

Business:
[Describe the company]

Sales process:
[Describe how leads move from first contact to payment]

Team:
[Who handles leads, sales, support, delivery]

CRM:
[Bitrix24 / amoCRM / HubSpot / other]

Lead sources:
[Website, Telegram, Avito, Instagram, ads, referrals, calls]

Required automations:
[List what should happen automatically]

Current problems:
[List problems]

## Output format

1. CRM pipeline structure
2. Stages and definitions
3. Required fields
4. Lead source tracking
5. Task automation
6. Notifications
7. Roles and permissions
8. Reports and dashboards
9. Implementation steps
10. Risks and common mistakes

## Rules

- Use simple language.
- Make the plan usable by a non-technical business owner.
- Include only necessary complexity.
